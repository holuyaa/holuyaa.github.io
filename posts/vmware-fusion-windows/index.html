    How to install Windows 11 on Arm Insider Preview

    September 17, 2022
    m1 mac
    m1 mac, vmware fusion 12 tech preview, Windows Insider Preview ARM64

    Prerequisite #

    VMware Fusion Public Tech Preview Download & Install #

    다운로드를 위해서는 회원 가입이 필요합니다.

    • vmware fusion 12 tech preview LINK

    다운로드 후에는 m1 mac에 설치합니다.

    Windows 11 on Arm Insider Preview Download #

    다운로드하기 위해서는 Microsoft 계정으로 Windows Insider Program을 register 해야됩니다.

    • Windows 11 on Arm Insider Preview LINK

    Windows 11 on Arm Insider Preview VHDX -> VMDK 변환 # 

    $ brew install qemu
$ qemu-img convert -O vmdk <다운로드 한 vhdx 파일명> <변경할 vmdk 파일명>

# 예를 들어
$ qemu-img convert -O vmdk Windows11_InsiderPreview_Client_ARM64_en-us_22598.VHDX Windows11_InsiderPreview_Client_ARM64_en-us_22598.vmdk

    Installation #

    Step 1: VMware Fusion 에서 vmdk 파일 선택 #

    1. VMware Fusion 에서 new 실행
    2. “Create a custom virtual machine” 선택 후 continue
    3. “Microsoft Windows -> Windows 11 64-bit Arm” 선택 후 continue
    4. UEFI 선택, UEFI Secure Boot 해제 후 continue
    5. “Only the files needed to support a TPM are encrypted. (.nvram, .vmss, .vmem, .vmx, .vmsn)” 선택
    6. Password & Verify 입력 후 continue
    7. “Use an existing virtual disk” 선택 후 “Choose virtual disk…” 클
    8. 변환 해 둔 vmdk 파일 선택

    Step 2: Windows 설치 #

    1. “Is this the right country or region?” 화면에서 Shift + F10 입력 후 터미널로 진입
    2. “OOBE\BYPSSNRO” 입력 후 엔터
    PS C:\Window\System32> OOBE\BYPASSNRO
    1. 네트워크 연결 화면에서 “I don’t have internet” 클릭
    2. “Continue with limited setup” 클릭
    3. VMWare Fusion에서 “Virtual Machine” -> “Reinstall VMWare Tools”
    4. 관리자 모드로 터미널 실행 후 아래 명령어 입력
    PS C:\Windows\system32> Set-ExecutionPolicy RemoteSigned
    1. 윈도우 탐색기 -> DVD Drive (D:) VMware Tools -> setup 파일 우클릭 -> Run with Powershell
    2. 재부팅

    Trouble Shooting #

    네트워크가 잘 동작하면 굳이 할 필요가 없지만 혹시 네트워크 연결이 안되서 인터넷이 안되는 분들은 아래 내용 확인

    접기/펼치기

    혹시 네트워크 연결이 안되서 인터넷이 안되는 분들은

    bcdedit /debug on
bcdedit /dbgsettings net hostip:10.0.0.1 port:55555

    Tips #

    • 맥 VMware Fusion 속에서 윈도우10 속도 올리는 법: LINK

    VMware Fusion 경로가 아래와 같음

    Applications/VMware\ Fusion\ Tech\ Preview.app/Contents/Library/

    실행할 때 / 종료할 때 Password를 입력해야 합니다.

    $ vmrun -vp <password> start <가상 윈도우 파일 절대 경로>
$ vmrun -vp <password> suspend <가상 윈도우 파일 절대 경로>
