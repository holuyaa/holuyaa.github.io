    How to setup hugo

    November 12, 2021
    hugo
    hugo

    Step 1: github에서 2개의 repository 생성 #

    • hugo 로 생성한 site의 소스코드를 위한 저장소
    • 렌더링된 버전의 Hugo 웹사이트를 포함할 .github.io 저장소

    private으로 blog, public으로 holuyaa.github.io를 생성했다. github id와 다른 abc.github.io로 생성하면 안된다. 꼭 github id와 동일 해야한다.

    Step 2: Install Hugo # 

    $ brew install hugo

    Step 3: Create a New Site # 

    # HOME directory로 이동
$ cd ~
$ mkdir blog
$ cd blog
$ hugo new site .

    Step 4: Add a Theme # 

    $ git init
$ git submodule add https://github.com/alex-shpak/hugo-book themes/book
# book theme의 config.toml 파일을 복사
$ cp themes/book/exampleSite/config.toml .

    복사된 config.toml에서 theme은 ‘book’으로 변경해줘야 한다.

    Step 5: Create Default Directories and Copy some files # 

    $ cd content
$ mkdir docs
$ mkdir posts
$ mkdir menu
$ cp ../themes/book/exampleSite/content/menu/index.md menu/
$ cp ../themes/book/exampleSite/content/posts/_index.md posts/
$ cd ..

    Step 6: Github 설정 # 

    # $HOME/blog로 이동
$ git remote add origin git@github.com:holuyaa/blog.git
# 아래 명령어 실행 후 public 디렉토리가 생성된다.
$ git submodule add -b master git@github.com:holuyaa/holuyaa.github.io.git public
$ cd public 
$ git add .
$ git commit -a -m "setup"
$ cd ..
$ git push
$ git add .
$ git commit -a -m "setup"
$ git push

    Step 7: Build static pages and Push # 

    $ cd $HOME/blog
# holuyaa.github.io 저장소에 올리려면 아래 명령어를 통해 build 해야한다.
$ hugo -t book
$ cd public
$ git add .
$ git commit -a -m "first commit"
$ git push

    Add some contents # 

    $ hugo new posts/my-first-post.md
